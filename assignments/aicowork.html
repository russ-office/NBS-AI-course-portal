Практическая ИИ-автоматизация

Создание работающей автоматизации реального рабочего процесса на своём компьютере с помощью Claude Desktop (Cowork).

День 3 (воскресенье) Индивидуально или в парах 40% итоговой оценки Требуется Claude Pro
Выбор варианта: Студент выбирает ОДИН из двух вариантов курсовой. Вариант Б (этот) требует подписку Claude Pro для работы с Desktop/Cowork. Если не готовы — выбирайте Вариант А: Технологическая стратегия.

Инструменты

  • Claude Pro (Desktop / Cowork) — единственный разрешённый инструмент автоматизации
  • Claude Skills — можно использовать готовые или создавать свои

Преподаватель проводит 30-минутную вводную по интерфейсу Claude Cowork перед началом работы.

Допустимые задачи

Уровень 1 — Файловая организация

  • Создать структуру папок для нового проекта по промпту
  • Переименовать группу файлов по единому шаблону (дата_тип_название)
  • Рассортировать файлы из одной папки по типам/датам/проектам
  • Организовать архив по определённой логике

Уровень 2 — Обработка документов

  • Извлечь данные из PDF/фото (счёт, накладная) в таблицу
  • Создать шаблон отчёта/письма, заполняемый по промпту
  • Обработать CSV/Excel: очистить, структурировать, добавить анализ
  • Сгенерировать документ на основе данных
Использование готовых Skills: Студент может найти и использовать готовый Claude Skill — это приветствуется. Логика «подумал → поискал аналог → адаптировал → создал сам» = лучший подход.

⚠️ Безопасность — обязательно прочитайте

ПРАВИЛА БЕЗОПАСНОЙ РАБОТЫ С ИИ-АВТОМАТИЗАЦИЕЙ
  1. Создай песочницу: папка AI_Workshop на рабочем столе. Работай ТОЛЬКО в ней.
  2. Копируй, не перемещай: нужны реальные файлы → КОПИРУЙ в песочницу. Никогда не работай с оригиналами.
  3. Не удаляй: никаких команд удаления. Только создание, копирование, переименование.
  4. Не трогай систему: не меняй настройки ОС, не устанавливай ПО без разрешения.
  5. Читай перед выполнением: если ИИ предлагает команду — прочитай и пойми. Не понимаешь → спроси преподавателя.
  6. Стоп-правило: что-то пошло не так → СТОП → зови преподавателя.

Формат сдачи

  1. Видео-демо (3–5 мин) или серия скриншотов с описанием шагов
  2. Описание (1 стр.): задача, инструменты, трудности, результат, экономия времени
  3. Экспорт чата с Claude (ссылка или PDF) — для проверки осмысленности работы
