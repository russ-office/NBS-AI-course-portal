Индивидуальная работа

Аналитика продуктового каталога

Извлечение структурированных данных из PDF-каталога мебельной фурнитуры, построение аналитических таблиц, диаграмм и отчёта с помощью ИИ.

Дни 1–2 Индивидуально Часть практической оценки ВСК Любая ИИ-модель

Каталоги на выбор

Выберите ОДИН каталог для анализа:

  • Вариант A — BOYARD (Katalog_FF_2024.pdf, ~340 стр.) — функциональная фурнитура: петли, направляющие, подъёмные механизмы, кухонное наполнение, опоры, замки, крепёж
  • Вариант B — BRASS (brass-catalog-2026-web.pdf, ~300 стр.) — лицевая фурнитура: ручки, крючки, петли, опоры, полкодержатели, замки, аксессуары для ванной

PDF-каталоги выдаются преподавателем в классе.

Инструменты

Подписка НЕ требуется. Можно использовать любую ИИ-модель: Claude, ChatGPT, Gemini, Copilot, Perplexity и др. Для таблиц и диаграмм — Google Sheets, Excel или ИИ-генерация.

7 аналитических задач (все обязательные)

  1. Иерархия категорий — полная структура каталога: категория → подкатегория → количество SKU
  2. Матрица «Тип × Цвет» — перекрёстная таблица: типы товаров × цвета/покрытия → количество позиций
  3. Матрица «Тип × Размер» — распределение товаров по типоразмерам внутри категорий
  4. Рейтинг категорий — столбчатая диаграмма: категории по количеству SKU + комментарий
  5. Анализ материалов — из каких материалов сделана фурнитура, распределение по категориям
  6. Тепловая карта ассортимента — heatmap: категории × параметр (цвет/размер/материал) → интенсивность
  7. «Белые пятна» — отсутствующие комбинации (тип × цвет × размер) + гипотеза: почему и есть ли бизнес-потенциал

Формат сдачи

  1. Отчёт (Фамилия_CatalogAudit_Report.pdf) — единый документ со всеми 7 заданиями: таблицы, диаграммы, комментарии + раздел «Рефлексия»

Допустимые форматы: PDF (предпочтительно), DOCX, PPTX. Максимум 50 MB.

Бонус +20%: Создайте интерактивный дашборд или HTML-страницу с визуализациями (минимум 3 из 7 заданий в интерактивном формате). Сдайте как URL через поле «Бонусный артефакт» в форме.

Рефлексия (обязательный раздел отчёта)

3–5 предложений: какой ИИ использовали, какие промпты давали, где ИИ ошибался, как проверяли точность данных.
