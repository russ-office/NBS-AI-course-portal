Индивидуальная работа

Аналитика продуктового каталога

Извлечение структурированных данных из PDF-каталога, построение аналитических таблиц, диаграмм и отчёта с помощью ИИ.

Дни 1–2 (пятница–суббота) Индивидуально 3–4 часа Подписка НЕ требуется

Исходные материалы

Все студенты работают с одним каталогом:

Swagelok — Pipe Fittings (MS-01-147, Rev T, May 2022): адаптеры, колена, тройники, муфты, заглушки, переходники, крестовины, ниппели. 23 страницы, английский язык.

📄 Скачать каталог (PDF, 2.4 MB)

Инструменты

  • ИИ-модель (любая): Claude, ChatGPT, Gemini, Copilot, Perplexity
  • Таблицы: Google Sheets, Excel или ИИ-генерация
  • Диаграммы: Google Sheets, Excel, Claude Artifacts, ChatGPT Canvas
  • Отчёт: Word, Google Docs, Canva, ИИ
Определение SKU: Допускаются два подхода. Подход A — базовые артикулы без учёта материала (~265 SKU). Подход B — каждый материальный вариант = отдельный SKU (~1850–1900 SKU). Зафиксируйте выбранный подход в отчёте.

7 аналитических заданий

1. Иерархия категорий

Извлечь полную структуру каталога: категория → подкатегория → количество SKU. Таблица с 3 уровнями вложенности.

2. Матрица «Тип × Материал»

Перекрёстная таблица: строки = типы фитингов, столбцы = 7 материалов, ячейки = количество позиций.

3. Матрица «Тип × Размер»

Распределение товаров по размерам резьбы (1/16"–1") внутри каждой категории. Выводы: какие размеры самые ходовые.

4. Рейтинг категорий

Рейтинг категорий по количеству SKU. Столбчатая диаграмма (bar chart) + комментарий: на чём фокусируется производитель.

5. Анализ материалов

Распределение материалов по категориям. Подсказка: материал определяется по префиксу артикула (SS-, S-, B-, 6Mo-, 625-, 825-, 2507-).

6. Тепловая карта ассортимента

Heatmap: строки = категории, столбцы = размеры, ячейки = количество SKU. Цветовое кодирование.

7. «Белые пятна» ассортимента

Выявить отсутствующие комбинации (тип × материал × размер × резьба). Минимум 5 «белых пятен» + для каждого гипотеза: почему нет и есть ли бизнес-потенциал.

Подсказки: Сравни доступность ISO/RT между категориями. Обрати внимание на Heavy-Wall (-10K) серию. Посмотри, все ли материалы доступны одинаково.

Формат сдачи

Маска имён файлов

Фамилия_CatalogAudit_ТипФайла.расширение

Примеры: Yegembayev_CatalogAudit_Report.pdf, Yegembayev_CatalogAudit_Tables.xlsx

Обязательные файлы

Отчёт (_Report.pdf) — единый документ со всеми 7 заданиями: таблицы, диаграммы, комментарии + раздел «Рефлексия» (3–5 предложений).

Допустимые форматы: PDF (предпочтительно), DOCX, PPTX. Максимум 50 MB. Языки: русский, английский, казахский.

Бонусный артефакт (+20% к оценке)

Интерактивный дашборд или HTML-страница с визуализациями (минимум 3 из 7 заданий в интерактивном формате). Сдаётся как URL.
